=====================
| Структура проекта |
=====================
- config (папка)
  - connect.php --> создаёт подключение к базы данных

- controllers (папка)
  - upload-files (папка) --> содержит все картинки автомобилей
  - addProduct.php --> обращается к базе данных и добавляет новый авто
  - addRental.php --> обращается к базе данных и добавляет бронь авто для пользователя
  - authUser.php --> обращается к базе данных и производит авторизацию пользователя
  - delProduct.php --> обращается к базе данных и удаляет авто из базы данных
  - delUserCart.php --> обращается к базе данных и удаляет корзину пользователя из базы данных
  - exitUser.php --> очищает сессию, тем самым сбрасывает авторизацию
  - filterProduct.php --> обращается к базе данных и ищет авто по заданному фильтру
  - getAllProduct.php --> обращается к базе данных и выводит все автомобили
  - getProduct.php --> обращается к базе данных и выводит всего один автомобиль по ID
  - getUserBasket.php --> обращается к базе данных и выводит корзину пользователя
  - regUser.php --> обращается к базе данных и производит регистрацию пользователя
  - searchProduct.php --> обращается к базе данных и выводит авто, исходя из названия в поиске
  - updateProduct.php --> обращается к базе данных и обновляет информацию об авто

- css (папка)
  - bootstrap-grid.min.css --> содержит в себе стили bootstrap (сетка)
  - reset.css --> сбрасывает стандартные стили браузера, чтобы на всех сайтах отображалось одинаково
  - style.css --> содержит все стили всего сайта

- fonts (папка)
  - Montserrat-Black.ttf --> Шрифт сайта

- img (папка)
  - image... --> содержит статические картинки (они НЕ берутся из базы данных)

- js (папка)
  - posts (папка) --> содержит все файлы для обращения к серверу
    - addItem.js --> получает значения полей, проводит их проверку (валидацию) и отправляет в файл 'addProduct.php'
    - authUsers.js --> получает значения полей, проводит их проверку (валидацию) и отправляет в файл 'authUser.php'
    - deleteItem.js --> получает значения полей и отправляет в файл 'delProduct.php'
    - filterItem.js --> получает значения полей, проводит их проверку (валидацию) и отправляет в файл 'filterProduct.php'
    - getItemAll.js --> получает все автомобили из файла 'getAllProduct.php' и выводит их на страницу
    - getUserCart.js --> получает все автомобили пользователя из файла 'getUserBasket.php' и выводит их в личном кабинете
    - regUsers.js --> получает значения полей, проводит их проверку (валидацию) и отправляет в файл 'regUser.php'
    - searchItem.js --> получает значения поля и отправляет в файл 'searchProduct.php'
  - main.js --> показывает или скрывает различные выезжающие меню при клике на кнопку

- pages (папка)
  - about.php --> Страница 'О компании'
  - admin-panel.php --> Страница 'Админ-панель'
  - autopark.php --> Страница 'Автопарк'
  - cabinet.php --> Страница 'Личный кабинет'
  - contact.php --> Страница 'Контакты'
  - price.php --> Страница 'Тарифы'

- authorization.php --> Страница 'Авторизация пользователя'
- index.php --> Страница 'Главная'
- registration.php --> Страница 'Регистрация пользователя'


============================
| Порядок загрузки проекта |
============================
1. Скачиваешь финальную версию проекта: https://github.com/Alex777234/car-rental-php
2. Скачиваешь SQL файл с базой данных: https://drive.google.com/file/d/1ZYbDb40fhuwJWTiqSkqpfS1BXPnRA25b/view?usp=sharing
3. Скачиваешь архив с картинками (если надо): https://drive.google.com/file/d/1vX0EerRbRkwP9i9eSArEC8M9_Do6mXZB/view?usp=sharing

4. Запускаешь Open Server
5. Открываешь папку localhost по следующему пути 'C://OpenServer/domains/localhost' или ПКМ на значок OpenServer -> Папка с проектами
6. Помещаешь файлы проета из пунтка 1 в эту папку
7. Открываешь PhpMyAdmin: ПКМ на значок OpenServer -> Дополнительно -> PhpMyAdmin
8. Создаёшь новую базу данных с именем 'car-rental'. Имя базы данных обязательно должно быть таким. Кодировку ставишь 'utf8_general_ci'
9. Открываешь вкладку 'Импорт' и выбираешь файл SQL из 2 пунтка, другие пункты не трогай
10. Вводишь в поисковой строке 'localhost' или ПКМ на значок OpenServer -> Мои проекты -> localhost
11. ВСЁ!
